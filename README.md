# Wireless Redstone (wlredstone)

A Fabric mod for Minecraft 26.2 that adds **wireless redstone**!

- **Wireless Transceiver** — a repeater-looking block you place on the ground. Place two of them.
- **Linker** — right-click the first transceiver, then the second one, and they get linked.
- The link is **two-way**: send a redstone signal into either transceiver and the other one outputs it.

## Building

Requires Java 25.

```
./gradlew build
```

The mod jar ends up in `build/libs/`.
